6.操作方法

6.1 concat()

concat()基于当前数组中的所有项创建一个新数组。具体步骤：首先创建数组的一个副本，然后将接受到的参数添加到这个副本的末尾，最后返回新构建的数组。没有给concat()传递参数时，它只是复制当前数组并返回副本。

注：concat()不会修改原数组

6.2 slice()

slice()可以基于数组中的一或多个项创建一个新数组。slice()方法可以接受一或两个参数，即要返回项的起始位置和结束位置。在只有一个参数的情况下，slice()方法返回从该参数指定位置到当前数组末尾的所有项。如果有两个参数，该方法返回起始和结束位置之间的项（左闭右开）。

注：slice()方法不会影响原数组

6.3 splice()

  • 删除：可以删除任意数量的项，只需指定2个参数：要删除的第一项的位置和要删除的项数。例如，splice(0,2)会删除数组中的前两项；

  • 插入： 可以向指定位置插入任意数量的项，只需提供3个参数：起始位置、0（要删除的项数）和要插入的项。例如，splice(2,0,”red”,”green”)会从当前数组的位置2开始插入字符串”red”和”green”。

  • 替换： 可以向指定位置插入任意数量的项，且同时删除任意数量的项，只需指定3个参数：起始位置、要删除的项数和要插入的任意数量的项，插入的项数不必与删除的项数相等。例如，splice(2,1,”red”,”green”)会删除数组位置2的项，然后再从位置2开始插入字符串“red“和”green”。

1.移动零

283. 移动零

题目： 给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

示例：

方法： 定义一个指针j指向数组最后一个非零元素，再从前面开始遍历数组，若遇到0，则通过冒泡的方式，将0移动到指针j的位置，并将j往前移动一个位置。

进阶方法： 定义一个指针index指向数组头部，再遍历数组，若元素值不为0，则将元素值与index位置的元素交换，保证非零元素在数组前端。

2.最大连续1的个数

485. 最大连续1的个数

题目： 给定一个二进制数组， 计算其中最大连续1的个数。

示例：

方法： 用len记录当前1的数量，max记录当前最大的1的数量。遍历数组，若nums[i]=1，则len++；若nums[i]!=1，则比较len与max的大小，若len>max，则将len的值赋给max，并将len置0。最后再比较max与len的大小，返回其中的最大值，这是为了避免[1,0,1,1]情况被漏掉。

方法二（双指针）： 定义指针i指向连续1的头部，j指向连续1的尾部。

给定一个数组 nums 代表了集合 S 发生错误后的结果。你的任务是首先寻找到重复出现的整数，再找到丢失的整数，将它们以数组的形式返回。

示例：

方法： 先将数组排序，再遍历数组。若出现相邻元素相等的，则该值为重复元素；若相邻元素的插值大于1，则可以找到丢失元素。

法二： 使用 hashmap 存储 nums中每个数字的出现次数。存储形式为 (num[i], count)当一个元素重复出现时，它的计数就加 1。

再检查 1 到 n 的每个数字在 map 中出现次数。如果一个数字在 map 中没有出现，它就是缺失数字。如果一个数字的出现了两次，它就是重复数字。

你的任务是找到与 nums 拥有相同大小的度的最短连续子数组，返回其长度。

示例：

方法：

定义三个map：

  1. mapCount：存储每个数组中每个元素出现的次数；

  2. mapFirstIndex：每个元素在数组中第一次出现时的下标；

6.有序矩阵中第K小的元素

378. 有序矩阵中第K小的元素

题目： 给定一个 n x n 矩阵，其中每行和每列元素均按升序排序，找到矩阵中第k小的元素。
请注意，它是排序后的第 k 小元素，而不是第 k 个不同的元素。

示例：

方法： 先用一个额外数组nums按顺序存储矩阵中的元素，再对数组进行排序，再返回数组中下标为k-1的值。

方法二： 二分查找，第k小的值一定在[left,right]中，其中left=matrix[0] [0]，right=matrix[row] [col]。定义mid=(left+right)/2，每次查找矩阵中比mid小的元素个数，如果mid<k，则left=mid+1，否则，right=mid。最后，当left=right时，right即为第k小的值。

给定一个 M x N 的矩阵，当且仅当它是托普利茨矩阵时返回 True。

示例：

方法： 每次将矩阵中的元素与右下方的元素相比较，若相等则继续，若不相等则返回false，最终返回true（既比较matrix[i] [j]与matrix[i+1] [j+1]是否相等）。

示例：

方法： 暴力会超时。问题就是寻找数组中的最大环，因为环外的元素不会访问到环内，所以可以将访问过的数组元素的值标为-1，下次不再访问，避免冗余。

10.搜索二维矩阵

240. 搜索二维矩阵 II

  • 每行的元素从左到右升序排列。
  • 每列的元素从上到下升序排列。

示例：

方法： 使用二分查找，查找矩阵的每一行。当行的第一个元素大于target时可跳出循环，当行的最后一个元素小于target时，可跳到下一行继续。

方法二： 可从矩阵右上角元素开始遍历矩阵，i为行下标（初始值为0），j为列下标（初始值为matrix[0].length-1）。当matrix[i] [j]的值大于target时，向左移动（j–），当matrix[i] [j]小于target时，向下移动（i++）,当matrix[i] [j] == target时，返回true。

示例：

限制：

  1. 不能更改原数组（假设数组是只读的）。
  2. 只能使用额外的 O(1) 的空间。
  3. 时间复杂度小于 O(n2) 。
  4. 数组中只有一个重复的数字，但它可能不止重复出现一次。

方法： 因为不能更改原数组，所以不可以对数组进行排序；因为只能使用额外的O(1)的空间，所以不可以采用Map()。因为数组的值在1到n之间，所以可根据数组元素的下标进行二分查找。思路如下：统计原始数组中小于等于中间数mid的元素的个数count，如果count严格大于mid，重复元素应该就在[left,mid]中。
